HTML

El lenguaje HTML, se emplea para crear las paginas web. Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a
otro y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nombre similar
